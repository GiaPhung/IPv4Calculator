# IPv4Calculator
A simple tool to subnetting IPv4 addresses.

##Features
* Some properties of IPv4 address.
* Subnetting IPv4 addresses.
* Subnet bitmap

##Requirements
* Windows 7 or later.
* .NET Framework 4.5.2 or later.
